**Codepth** is a beginner-friendly, web-based Python tutorial platform that teaches complete Python fundamentals in **Hinglish (Hindi + English)** — making programming highly accessible for Indian learners.

---

## ✨ What Makes It Unique

Every topic includes a **live in-browser code editor** — a Jupyter-style interactive cell where learners can write, edit, and run Python code directly in the browser with **zero setup required**. Powered by a custom Python-to-JS transpiler built in vanilla JavaScript.

## 🚀 Key Features

- 🖊️ **Interactive Code Cells** — Jupyter-style editor on every page with line numbers, Tab indentation & Ctrl+Enter to run
- ⚡ **In-Browser Python Runner** — Custom JS engine handles `print()`, f-strings, loops, OOP, exceptions, RegEx and more
- 📁 **Virtual File System** — Simulates real file I/O for the File Handling module
- ❌ **Custom Exception Engine** — Full Python exception hierarchy in the browser
- 📱 **Responsive Design** — Works on desktop and mobile
- 🔀 **Smooth Navigation** — Sidebar + Previous/Next page buttons
- 🇮🇳 **Hinglish Content** — Real-world examples explained in Hindi + English
- 🌐 **No Backend** — Fully static, deploy anywhere

## 🖥️ Getting Started

Since this is a fully static project, no installation is needed.

**Option 1 — Open directly:**
```bash
# Clone the repo
git clone https://github.com/your-username/codepth.git

# Open in browser
open index.html
```

**Option 2 — Live Server (recommended for development):**
```bash
# Using VS Code Live Server extension
# Right-click index.html → "Open with Live Server"
```

**Option 3 — Deploy online:**
> Deploy directly to **GitHub Pages**, **Netlify**, or **Vercel** — no configuration needed.
