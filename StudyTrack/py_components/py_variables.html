Python me variables kya hote hai ?


Sabse pehele aap log ye image dekho :- 



Simple shabdo me: 👉 Variable ek container ki tarah hota hai jo data ko store karta hai.

Is image me variable ko ek jar (container) ke example se samjhaya gaya hai. Jaise har jar ke andar koi cheez hoti hai jaise sugar, chilli ya oil, waise hi programming me variable ek container hota hai jisme data store hota hai. Jar ka naam variable name hota hai aur jar ke andar ki cheez data ya value ko represent karti hai.


Abb seekho variable kaise banaya jaata hai.

Python
In [1]:
1
Output

Abb iss code me a = 10 and name = 'Bill Gates' aisa kuch likha hai. Iska bada simple matlab hai ki memory me 2 jaghe banri hai jisme 10 aur 'Bill Gates' stored hai par uss jaghe ka naam a aur name hai.

Note : Python me yaad rakhna '=' ke left side jo likha hai usme '=' ke right side wali cheeze store hoti hai.


( a = 10 ka matlab hai 'a' me '10' rakha hai )

Variable banane ke kuch rules :

  • Variable ka naam letter (a–z, A–Z) ya underscore (_) se start hona chahiye
  • Variable ka naam number se start nahi ho sakta.
  • Variable name me sirf letters, numbers aur underscore (_) use ho sakte hain.
    Koi special charcter jaise '$@#' use nahi karna
  • Variable name me space allowed nahi hota.
  • Variable ka naam Python keywords (jaise if, for, while) nahi ho sakta.
  • Python case-sensitive hai, isliye name aur Name alag variables hote hain.

Python
In [1]:
1
Output
