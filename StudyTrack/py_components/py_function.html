Python me Functions kya hote hain?

Function ek reusable block of code hai jo ek specific kaam karta hai. Ek baar function define karo, aur jab chahein baar baar call karo — code dobara likhne ki zarurat nahi. Jaise ek machine jo ek specific kaam karna jaanti hai — button dabao aur kaam ho jaata hai.

Functions kyun use karte hain :-

  • Reusability — ek baar likho, baar baar use karo
  • Readability — code clean aur samajhne mein aasaan hota hai
  • Modularity — bade program ko chhote parts me todna
  • Less Errors — ek jagah change karo, sabko faayda
