Python me For Loop kya hota hai?

For loop ka use tab hota hai jab hume kisi sequence ke har ek item par baar baar koi kaam karna ho — jaise list ke har element ko print karna, string ke har character ko check karna, ya kisi range me kaam karna. While loop me hum khud counter manage karte hain, lekin for loop automatic ek ek item par jaata hai.

Is page me yeh sab sikhenge :-

  • 1. Basic for loop
  • 2. range() function ke saath
  • 3. String pe for loop
  • 4. List, Tuple, Set pe for loop
  • 5. Dictionary pe for loop
  • 6. break aur continue
  • 7. Nested for loop
  • 8. for-else

Note :- For loop Python me kisi bhi iterable (list, string, tuple, dict, set, range) pe chal sakta hai. Har iteration me variable automatically next item le leta hai.


1. Basic For Loop

For loop ek sequence ke har ek item ko ek ek karke visit karta hai. Jab tak sequence me items hain, loop chalta rehta hai.

Syntax
 
for variable in sequence:
    # har iteration me variable ka value change hota hai
