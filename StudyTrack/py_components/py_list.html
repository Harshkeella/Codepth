Python me List kya hoti hai?

List Python ka ek ordered collection hai jisme hum multiple values ek saath store kar sakte hain. List me kisi bhi data type ke values ho sakte hain — numbers, strings, booleans, ya dusri lists bhi!

List ki 4 main khasiyatein hain :-

  • Ordered — items ek fixed order me rehte hain
  • Mutable — items baad me change, add ya remove ho sakte hain
  • Indexed — har item ka ek index hota hai (0 se shuru)
  • Duplicates allowed — same value kai baar aa sakti hai
