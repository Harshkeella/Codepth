Python Install kaise kare ?


Step 1 : Python official website pe jaao.

👉 :- https://www.python.org

  • 1. Download Python pe click karo ye .exe file install kardega.
  • 2. Installer ko run karo and ADD Python to PATH isko bhoolna nai.
  • 3. 1-2 min me setup was successful aisa message aa jayega.
  • 4. Installation verify kar lena command prompt pe jaake python --version ye run karna.
  • 5. Test kar lena terminal(cmd) pe jaake python run karlo.
  • 6. uske baad print('Hello World') agar ye print ho gaya to samjo sahi chalre ho.

Step 2 : Code Editor Install karlo.

  • 👉 Visual Studio Code:- Download

  • 👉 PyCharm :- Download


Step 3 : Jisko Install nahi karna vo online karlo.

  • 👉 Google colab:- Go!
