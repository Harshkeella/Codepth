Python me Regular Expressions (RegEx) kya hai?

Regular Expression (RegEx) ek pattern hai jo strings me specific text dhundhne, check karne ya replace karne ke kaam aata hai. Jaise email validate karna, phone number check karna — yeh sab RegEx se bahut aasaan ho jaata hai. Python me re module use hota hai.

Is page me yeh sab sikhenge :-

  • 1. re module aur basic functions
  • 2. Meta characters
  • 3. re.search() aur re.match()
  • 4. re.findall()
  • 5. re.sub()
  • 6. re.split()
  • 7. Groups aur Practical Examples

1. re Module aur Basic Functions

FunctionKaam
re.match(pat, str)String ke shuru me pattern dhundho
re.search(pat, str)String ke kahi bhi pattern dhundho
re.findall(pat, str)Saare matches ki list do
re.sub(pat, repl, str)Pattern ko replace karo
re.split(pat, str)Pattern pe split karo
re.compile(pat)Pattern ko compile karo
