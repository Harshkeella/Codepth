Python me Keywords kya hote hai ?


Python ke developers ne kuch shabdon ko pehle se hi Python ke liye reserve kar diya hai, kyunki inka programming language me special aur fixed meaning hota hai. In reserved shabdon ko keywords kaha jata hai, aur inhe variable name, function name ya identifier ke roop me use nahi kiya ja sakta.


Note :- Python me total 35 Keywords hai aur 3 Literals hote hai.


Literals :-

  • True
  • False
  • None

True , False aur None ko literals isliye kaha jaata hai kyunki ye fixed values ko directly represent karte hain aur inki value program me change nahi hoti.

Agar aapko vo 35 keywords dekhne hao to bas apne code editor pe jaake ye commands likhe.
