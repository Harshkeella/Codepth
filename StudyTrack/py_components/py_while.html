Python me While Loop kya hota hai?

Jab hume ek kaam baar baar karna ho — jab tak koi condition True rahe — tab hum while loop use karte hain. Jaise ek cricket match me batting tab tak chalti hai jab tak batsman out na ho jaye — exactly waise hi while loop kaam karta hai.

Is page me yeh sab sikhenge :-

  • 1. Basic while loop
  • 2. while loop me counter
  • 3. break — loop beech me rokna
  • 4. continue — ek iteration skip karna
  • 5. while-else
  • 6. Infinite loop — aur use kaise sambhalen

Note :- While loop me condition ka False hona zaroori hai warna loop kabhi band nahi hoga — ise Infinite Loop kehte hain. Isliye loop ke andar counter ya update zaroor likhein.


1. Basic while Loop

while loop tab tak chalta rehta hai jab tak condition True ho. Jaise hi condition False hoti hai, loop band ho jata hai.

Syntax
 
while condition:
    # yeh code tab tak chalega jab tak condition True ho
    # counter update karna mat bhoolna!
