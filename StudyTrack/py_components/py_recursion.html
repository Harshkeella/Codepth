Python me Recursion kya hota hai?

Recursion ek aisi technique hai jisme function khud apne aap ko call karta hai. Jaise aap ek darwaza kholte hain aur andar jaate hain, phir aur ek darwaza, phir aur — jab tak aakhri kamra na aaye. Wapas aate waqt har darwaza band karte jaate hain. Exactly yahi Recursion me hota hai.

Har recursive function me 2 zaroori cheezein honi chahiye :-

  • Base Case — woh condition jab recursion rukti hai (warna infinite loop)
  • Recursive Case — function khud ko chhote problem ke saath call karta hai

1. Recursion Kaise Kaam Karta Hai?

Recursion me function har baar chhoti problem ke saath apne aap ko call karta hai, jab tak base case na aa jaye. Phir sab answers wapas aate hain (unwind).

Syntax
 
def recursive_function(problem):
    if base_case:          # yahan rukna hai
        return answer
    else:
        return recursive_function(smaller_problem)  # apne aap ko call
