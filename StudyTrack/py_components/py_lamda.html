Python me Lambda Function kya hota hai?

Lambda function ek anonymous function hai — yaani bina naam ka function. Isko ek hi line me likha ja sakta hai. Yeh tab useful hota hai jab aapko chhota sa function sirf ek baar ya thodi der ke liye chahiye ho aur poora def likhna unnecessarily lamba lage.

Lambda ki 3 main khasiyatein hain :-

  • Anonymous — naam nahi hota (usually)
  • One-liner — sirf ek expression likh sakte hain
  • Auto return — return likhna nahi padta, automatically return hota hai

1. Lambda Syntax aur Basics

Lambda ka syntax bahut simple hai — lambda keyword, phir parameters, phir colon (:), phir expression (jo automatically return hota hai).

Syntax
 
lambda parameters : expression

# Example:
lambda x : x * 2
